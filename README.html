SEMESTRAL PROJECT - CHESS

Client/server application using JavaFX

Running

Running client

mvn exec:java -Dexec.arguments=client

mvn exec:java -Dexec.arguments=client

How-to

Run client and select new game, that’s all the functionality right now.
